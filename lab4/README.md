## Анализ результатов

### 1. Зависимость от размера блока

| Размер матрицы | Оптимальный блок | Макс. ускорение | Макс. GFLOPS |
|----------------|------------------|-----------------|--------------|
| 200 | 16×16 | 1019× | 131.5 |
| 400 | 16×16 | 1366× | 1781.5 |
| 800 | 16×16 | 784× | 1022.9 |
| 1200 | 16×16 | 758× | 989.3 |
| 1600 | 32×32 | 1028× | 1332.7 |
| 2000 | 32×32 | 1737× | 2251.8 |

**Вывод:** Для малых матриц (≤1200) оптимален блок 16×16, для больших (≥1600) — 32×32.

### 2. Зависимость от размера матрицы

- **CPU:** Время растет как O(N³) — от 130 мс до 132 секунд
- **GPU:** Время растет медленнее — от 0.13 мс до 76-168 мс
- **Ускорение:** Растет с размером задачи (от 455× до 1737×)

### 3. Производительность

- **Максимальная производительность GPU:** 2252 GFLOPS (2.25 TFLOPS) на матрице 2000×2000 с блоком 32×32
- **Минимальное время GPU:** 0.127 мс (200×200, блок 16×16)

### 4. Эффективность использования GPU

| Размер | Эффективность (относительно пика) |
|--------|----------------------------------|
| 200 | 5.8% |
| 400 | 79.1% |
| 800 | 45.4% |
| 1200 | 43.9% |
| 1600 | 59.2% |
| 2000 | 100% |

## Выводы

1. **CUDA обеспечивает колоссальное ускорение** — до **1737×** по сравнению с CPU

2. **Оптимальная конфигурация:**
   - Матрицы ≤1200: блок **16×16**
   - Матрицы ≥1600: блок **32×32**

3. **Пиковая производительность GPU:** **2.25 TFLOPS** на Tesla T4

4. **Масштабируемость:** Ускорение растет с размером задачи

5. **Верификация:** Все результаты проверены, ошибок нет
